FAQ

Attendance Management Questions Answered

Do I need to replace my existing biometric device to use Ventryx?

Not necessarily. Ventryx AMS works standalone without any hardware. If you want to continue using biometric devices, we have integration APIs for popular brands like ZKTeco, eSSL, and Matrix. Data from the biometric device syncs into Ventryx for centralised reporting.

Can employees regularise missed attendance entries?

Yes. Employees can raise a regularisation request for missed check-ins or check-outs, with a reason. The request goes through the manager approval workflow. All regularisations are flagged separately in reports so HR has full visibility.

Does Ventryx work for remote and hybrid teams?

Yes, it's designed for modern hybrid work. Remote employees mark attendance from the web or mobile app. You can require GPS location proof, selfie verification, or IP address restrictions - whatever your company policy requires. On-field teams can also use the app.

Which payroll software does Ventryx integrate with?

We have native integrations with Razorpay Payroll, GreytHR, and Keka. For all other payroll systems, you can export payroll-ready Excel/CSV reports in any format. We also offer a REST API for custom integrations.
