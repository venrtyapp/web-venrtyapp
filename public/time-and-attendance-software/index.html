FAQ

Time and Attendance Questions Answered

What is time and attendance software?

Time and attendance software tracks when employees start and end work, calculates total hours worked, overtime, late arrivals, and absences - then feeds this data into payroll. Modern systems like Ventryx do this via mobile app, web, and kiosk, replacing traditional biometric punch-in machines with a more flexible and accurate cloud-based approach.

How accurate is mobile time and attendance tracking?

Ventryx uses GPS location verification and optional selfie capture to ensure accuracy. The system timestamps every check-in to the second and stores this data with GPS coordinates. This creates a more verifiable record than biometric punch-in, which only confirms presence at the device location - not the actual work location for field employees.

Can I track overtime automatically?

Yes. Ventryx automatically calculates overtime based on your configured shift rules. You define the standard working hours per shift, and the system calculates anything beyond that as overtime. Managers see overtime alerts in real-time, and reports show overtime by employee, department, or shift - ready for payroll processing.

Does Ventryx time tracking integrate with payroll software?

Yes. Ventryx exports payroll-ready data compatible with GreytHR, Keka, Razorpay Payroll, and all major Indian payroll systems. Custom Excel/CSV export is available for any other system. Enterprise customers can use the Ventryx REST API for real-time payroll integration.

Does Ventryx work for manufacturing and factory shift workers?

Yes. Ventryx handles three-shift A/B/C operations, night shift differentials, rotational rosters, and biometric device integration for factory gates. It tracks shift-wise attendance separately, generates Factories Act compliance reports, and manages contractor labour tracking independently from permanent staff.
